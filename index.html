Informações gerais

O template usa bootstrap 5, font-awesome 5 e jquery 3.6. Para animações ele possui linkado animate.css

O template utiliza de forma legado o print.js e a paginação dinâmica. Foi acrescentada a possibilidade de adicionar um print.css e um leitor.css customizado, bastando incorporar o caminho no html usando uma tag de script. No final do conteúdo de exemplo temos comentado como proceder.

O arquivo icones.css possui o css pra caixa com icone, bem como a forma que é para adicionar novos icones se necessário.

Para quiz simples o template possui incorporado o esquema da bru(ver no conteudo linkado ou no arquivo js como é a aplicação)

Possui um player de audio customizavel próprio
